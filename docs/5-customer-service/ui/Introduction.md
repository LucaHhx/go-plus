# 客服系统 -- 前端设计说明

## 设计概述

客服系统包含两个部分: 在线客服底部面板 + 社交媒体链接（已包含在首页页脚）。设计轻量，不需要独立页面。

## 在线客服面板

### 触发方式

侧边菜单 "Live Support" 点击触发。

### 面板行为

- 从底部滑入，高度 85vh
- 背景遮罩 rgba(0,0,0,0.6)
- 300ms ease-out 动画
- 顶部 16px 圆角
- 点击遮罩或 X 按钮关闭

### 对话 UI

- 客服消息: 左对齐，灰色气泡 (#2A2D2D)，左上角 4px 圆角
- 用户消息: 右对齐，品牌色气泡 (#24EE89)，深色文字，右上角 4px 圆角
- 气泡最大宽度 75%
- 客服头像: 品牌色透明背景 + 聊天图标
- 打字中: 三个灰色圆点 bounce 动画

### 输入区

- 圆角输入框 (pill 形状, 40px 高)
- 右侧品牌色圆形发送按钮
- 空输入时发送按钮变灰色

## 社交媒体链接

已包含在首页页脚 Community 区域，无需单独实现。

## 前端注意事项

1. 第一期客服面板为 Mock UI，消息不需要实际发送
2. 后续接入第三方客服 SDK 时替换面板内容
3. 面板使用 `position: fixed`，覆盖在当前页面之上
4. 对话区域需要自动滚动到最新消息
5. 社交媒体链接使用 `target="_blank"` 新窗口打开
6. 面板打开时禁止背景页面滚动 (`overflow: hidden` on body)
