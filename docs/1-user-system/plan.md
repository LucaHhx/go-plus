# 用户系统

> 提供用户注册、登录、身份验证等基础账户功能，支持手机号和 Google 登录

## 目标

1. 用户可以通过手机号 + OTP 注册账户
2. 用户可以通过 Google 账号快速注册/登录
3. 用户可以通过手机号+密码或 OTP 登录
4. 新注册用户获得欢迎奖励 (100 Bonuses)
5. 注册流程中用户可选择赠送游戏 (Aviator 或 MoneyComing 免费旋转)

## 范围

**包含:**

- 手机号注册 (手机号 + 密码 + OTP 验证)
- Google 第三方登录
- 手机号 + 密码登录
- OTP 快捷登录
- 注册欢迎奖励发放
- 注册赠送游戏选择
- 用户会话管理 (登录态维持/过期/登出)
- 用户基本信息查看 (登录后可查看自身手机号、余额等基本信息)

**不包含 (不属于本需求范围):**

- VIP 等级系统 (归属 VIP 系统需求)
- 推广/邀请码关联 (归属推广系统需求)
- 多市场手机号格式 (首发仅支持 +91 印度号码)

## 核心用户场景

- 场景 A: 新用户首次访问平台，点击 "Join Now" 绿色按钮（样式与设计图纸一致），弹出注册表单，布局和视觉效果与 UI 设计图纸 (merge.html) 完全一致，输入手机号和密码，接收 OTP 短信验证，选择欢迎赠送游戏，完成注册并获得 100 Bonuses
- 场景 B: 用户选择 Google 登录，授权后自动创建账户或关联已有账户，首次登录同样获得欢迎奖励
- 场景 C: 已注册用户通过手机号 + 密码登录，登录表单样式与设计图纸一致，进入个人主页
- 场景 D: 用户忘记密码或偏好快捷方式，使用 OTP 验证码直接登录

## 背景

用户系统是整个平台的基础设施，所有其他功能模块（游戏大厅、钱包、VIP 等）都依赖用户身份。根据 UI 设计图纸，注册入口为顶部 "Join Now" 绿色按钮，登录入口为 "Log In" 透明按钮。注册流程简洁，强调快速进入游戏体验。
