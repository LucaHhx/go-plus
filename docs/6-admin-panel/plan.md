# 管理后台

> 提供运营团队管理平台的后台系统，覆盖用户管理、游戏管理、交易管理等基础运营功能

## 目标

1. 管理员可以查看和管理注册用户 (查看/搜索/禁用/详情)
2. 管理员可以管理游戏和供应商 (上下架/分类/排序)
3. 管理员可以查看和处理交易 (充值/提现审核)
4. 管理员可以查看平台运营数据 (Dashboard 概览)
5. 管理员可以管理 Banner 和首页内容

## 范围

**包含:**

- 管理员登录与权限控制
- 仪表盘 (Dashboard): 关键指标概览 (注册数/活跃数/充值额/提现额)
- 用户管理: 列表/搜索/查看详情/禁用/启用
- 游戏管理: 游戏列表/上下架/分类管理/供应商管理
- 交易管理: 充值记录/提现记录/提现审核
- 内容管理: Banner 管理/首页内容配置
- 系统配置: 支付渠道配置/基础参数设置

**不包含 (不属于本需求范围):**

- 促销活动管理 (归属促销系统需求)
- VIP 等级管理 (归属 VIP 系统需求)
- 奖池配置 (归属奖池系统需求)
- 抽奖/任务/推广管理 (归属对应系统需求)
- 数据分析/报表系统

## 核心用户场景

- 场景 A: 运营人员登录管理后台，在 Dashboard 查看今日注册数、活跃数、充值总额、提现总额等关键指标
- 场景 B: 运营人员收到用户投诉，在用户管理中搜索该用户手机号，查看用户详情 (注册时间/余额/交易记录)，进行处理
- 场景 C: 新游戏供应商接入，运营人员在游戏管理中添加供应商，批量导入游戏，设置分类和排序，上架发布
- 场景 D: 用户发起提现，在交易管理中出现待审核提现单，运营人员核实后批准或拒绝
- 场景 E: 运营人员需要更换首页 Banner，在内容管理中上传新 Banner 图片，设置排序和跳转链接

## 背景

管理后台是平台运营的必备工具，聚焦基础运营能力: 用户管理、游戏管理、交易管理。随着后续客户端功能的增加，管理后台也会相应扩展。管理后台的用户是内部运营团队，非面向玩家。
